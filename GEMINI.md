# THE FRIDGE

## Project Overview
A smart pantry management Flutter app with Spatial Glassmorphism design.

## Tech Stack
- **Framework:** Flutter
- **Language:** Dart
- **State Management:** Riverpod
- **Navigation:** GoRouter
- **Database:** Drift (SQLite)
